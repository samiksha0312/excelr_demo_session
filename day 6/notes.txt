anchor tag:
-----------
- <a> tag - used to create a link, which can navigate to another page
- <a> tag attributes:
1. href - used to specify the url of the linked page
2. target - used to sepcify the target of the linked page
   target="_blank" - opens the linked page in a new table
   target="_self" - opens the linked page in the same tag
   target="_parent" - opens the linked page in the parent frame
   target="_top" - opens the linked page in the top frame
3. rel - used to specify the relationship of the linked page
4. download - used to specify the download attribute of the linked page.

iframe tag:
-----------
- <ifram> tag - used to create an inline frme, which can navigate to another page
- <iframe> tag attribute - 
 1. src - used to specify the url of the linked page 
 2. width- used to specify the width of the frame
 3. height - used to specify the height of the frame
 4. frameborder - used to specify the border of the frame
 5. scrolling - used to specify the scrolling of the frame

embed tag:
----------
     - <embed> tag - used to embed a file in the web page 
     - <embed> tag attributes:
        1. src - used to specify the url of the file
        2. type - used to specify the type of the file
           audio - audio/mp3
           video - video/mp4
           image- image/jpg
        3. width - used to specify the width of the file
        4. height - used to specify the height of the file
    Note: it can be embed image, video, audio, pdf etc in the web pages but won't support controls

image tag:
-----------
    - <img> tag - used to add an image in the web page 
    - <img> tag attributes :
      1. src - used to specify the url of the file
      2. width - used to specify the width of the image
      3. height - used to specify the height of the image 
      4. alt - used to specify alternative text of the image
      5. title - used to specify the title of the image

audio tag: 
-----------
    - <audio> tag - used to add an audio in the web page
    - <audio> tag attributes: 
    1. src - used to specify the url of the audio
    2. controls- used to specify the controls of the audio
    3. autoplay - used to specify the autoplay of the audio
    4. loop - used to specify the loop of the audio

video tags:
-----------

     1. src - used to specify the url of the video
     2. controls - used to specify the controls of the video
     3. autoplay - used to specify the autoplay of the video
     4. loop - used to specify the loop of the video
     5. Poster - used to specify the poster of the video

marquee tag:
-----------
     -<marquee> - marquee tag is used to scroll content in the web page
     -<marquee> tag attributes:
      1. direction - used to specify direction of the scroll
        direction="left" - scroll from right to left(default)
        direction="right" - scroll from right to right
        direction="up" - scroll towards up
        direction="down" - scroll towards down
      2. behavior - used to specify the behavior of the scroll
         behavior="alternate" - scroll back and forth
         behavior="scroll" - scroll indefinitely(default)
         behavior="slide" - scroll once
      3. scrollamount - used to specify speed of the scroll
      4. scrolldelay - used to specify delay of the scroll
      5. loop - used to specify the number of times the scroll repeats

Semantic tags:
--------------
  - semantic tags are used to provide meaning to the web pages
  - semantic tags are used to improve the accessibility, searchability, and usability of the web pages
  - Introduced in HTML5
