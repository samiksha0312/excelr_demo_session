Selector:

It is used to select html elements for Styling
  • element selector
  • child selector
  • classs selector
  • attribute Selector
  • grouping selecor
  • Universal Selector
  • combined selector
    • descendent selector
    • child selector
    • adjacent sibling selector
    • general sibling selector

Element selector:

It is used to select all the elements by their tag name
            Syntax:          tagname{
                                   property:value;
                                    }

ID selector:

it is used to select elements based on their id

id is an attribute used in opening tag

id should be unique

id selector indiccated with #

             Syntax:      #idname{
                               property: value ;
                               }

class selector:

It is used to select the elements by their class

class is an attribute used in opening tag

class name should be duplicate

class selector indicated with .

            syntax:     .classname{
                           property:value;
            }

Grouping selector:

It is used to select elements as grouping

grouping selector indicated with ,

it is used to apply same style to multiple elements

            Syntax:       selector1,selector2,selector3{
                                property:value;
            }

Combined selector

It is used to combine multiple selectors

It is used to apply same style to multiple elements

Descendent selector

it is used to select elements that are descendents of a specified element

used to select both direct and indirect child elements

descendent selector indicated with a "space"

            syntax:         parent child{
                           property:value
                         }

child selector

it is used to select the elements that are direct children of a specifiied Element

Used to select the only direct child elements

child selector indicated with" > "

            Syntax:         parent>child{
                              property:value;
                             }

Adjacent Selector

It is used to select elements that are adjacent sibling of a specified element

used to select only the next sibling element

adjacent sibling selector indicated with "+" 

 
            Syntax:     element1=element2{
                         property:value;
            }

General Sibling selector

It is used to select the elements that are general sibings of a specified element

used to select all sibling elements

general sibling selector indicated with " ~ "

        syntax:   element1~element2{
            property:value;
        }

Universal Selector

It is used to select all elements

Universal selector indicated with *

        syntax:       *{
                        property: value;
                    }
        Note : it is recommended to have 
        padding :0;
        margin:0;
        box-sizing: border-box;
        content-box-it is used to set the box model by default
                   -it calculates the width and height of an element
                    including padding and border
                    ex: width: 100px;
                        height: 100px;
                        padding: 10px;
                        border: 2px solid black;
                        box-sizing: content-box;
                    calculation:
                        width: 100px + 10px + 10px+2px+2px = 124px;
                        height: 100px + 10px + 10px+2px+2px = 124px;

     box-sizing: border-box-it is used to set the box model by manually
                -it calculates the width and height of an element 
                including padding and border and provides the exact width and height
                ex: width: 100px;
                    height: 100px;
                    padding: 10px;
                    border: 2px solid black;
                    box-sizing: border-box;
                calculation:
                    width: 100px + 10px + 2px = 100px;
                    height: 100px + 10px + 2px = 100px;
